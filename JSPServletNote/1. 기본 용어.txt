1. 서버(Server)
    1) 웹에서 서비스를 제공하는 컴퓨터 시스템 

2. 웹 서버(Web Server)    
    1) 사용자로부터 http를 통해 요청(request)을 받거나, 
       웹 컨테이너가 전달해 준 결과물을 정적인 페이지로 생성하여
       사용자에게 응답(response)해주는 소프트웨어임
    2) 웹 페이지는 주로 HTML, CSS, JS 등으로 구성됨      

3. 웹 컨테이너 (Web Container)      
    1) 웹 서버가 전달해준 요청을 기초로 동적인 페이지를 생성하여 
       웹 서버로 돌려줌.
    2) 동적인 페이지
        - 사용자마다 다른 결과로 응답할 수 있기 때문임 
        - 예) 로그인 
            - 로그인한 아이디를 보고 각자에 맞는 페이지를 구성해 보내주는 것임    

4. WAS (Web Application Server)
    1) 웹 애플리케이션이 실행 될 수 있는 환경을 제공하는 소프트웨어임
    2) 웹 서버와 웹 컨테이너를 포함한 개념 
    3) 예) 톰캣(Tomcat), 웹로직, 웹스피어 등   

5. HTTP(HyperText Transfer Protocol) / HTTPS (HTTP Secure)             
    1) www(World Wide Web)에서 웹서버와 사용자 사이의 통신을 위해 사용하는 통신 프로토콜임
    2) 사용자가 요청하면 웹 서버가 응답하는 단순한 구조의 프로토콜 
    3) HTTPS는 암호화된 HTTP임 

6. 프로토콜(Protocol)
    1) 네트워크 통해 컴퓨터들이 정보를 주고받는 절차 혹은 통신 규약.
    2) 예) HTTP, FTP, SMTP 등    

7. 포트 (Port) 
    1) 컴퓨터 사이에서 데이터를 주고 받을 수 있는 통로 
    2) 인터넷에서는 IP 주소를 통해 서버 컴퓨터의 위치를 파악 
    3) 그런 다음 그 컴퓨터가 제공하는 특정 서비스는 포트 번호를 통해 알 수 있음 
    4) 인터넷 상의 모든 서비스는 IP주소와 함께 포트 번호까지 지정해야 제대로 요청을 전달할수 있음
