0. 정보화 사회 이해
    1) 농업혁명 => 산업혁명 => Big data, AI(Machine Learning, Deep Learning)
    2) 데이터(data)
        - 정보의 원재료
        - 이진(binary)과 같은 형태로 바뀌어진 자료 
        - 데이터 표현 단위 
            - 비트(Bit) : 0, 1
            - 바이트(Byte) : 1 byte = 8 bit
            - 킬로바이트(KB, KiloByte) : 1KB = 1024 byte = 2^10 byte 
            - 메가바이트(MB, MegaByte) : 1MB = 1024 Kbyte = 2^20 byte 
            - 기가바이트(GB, GigaByte) : 1GB = 1024 Mbyte = 2^30 byte 
            - 테라바이트(TB, TeraByte) : 1TB = 1024 Gbyte = 2^40 byte 
    3) 정보(information)
        - 가공된 데이터    
        - 어떤 사물에 대한 소식이나 자료      

1. 메인 보드 
    1) 다양한 물리적인 장치(device)를 장착할 수 있는 인쇄회로기판(PCB)임.
    2) 컴퓨터의 CPU, 메인 메모리, 하드 디스크(보조기억장치)등을 장착함.
    3) 다양한 컴퓨터 부품을 연결할 수 있는 인터페이스를 제공함.
    4) Motherboard라고도 불림. 
    5) 데이터의 전달 통로가 디자인 되어있는 메인 보드 

2. GPU (Graphics Processing Unit)
    1) 그래픽 관련 연산을 빠르게 처리하는 연산 장치임 
    2) 인공지능(딥러닝), 고화질 게임, 무인 자동차 등에서 많이 쓰임 
    3) 컴퓨터 그래픽/영상 처리에 유용하게 활용됨.
    4) 대규모 행렬 및 벡터 연산을 처리하는데 뛰어난 성능을 발휘함.    

3. 보조 기억 장치 
    1) 하드 디스크 : 전통적으로 많이 사용된 보조 기억 장치임.
        - 큰 용량의 프로그램을 장기간 저장하기 위해 효과적으로 사용 가능함 
    2) SSD : 하드 디스크보다 빠르고 전력 소비가 적음.   
        - 최근 빠르면서 대용량 조건을 만족하게 되어 실질적으로 많이 쓰이고 있음.
    3) 장점 
        - 전원이 차단되어도 내용이 유지되고, 저장 용량이 큼 
       단점
        - 보조기억장치는 메인 메모리보다 속도가 느림                    

4. 메인 메모리 (Main Memory)
    1) 주기억장치라고 부름 
    2) 일반적인 개인용 컴퓨터에서 RAM (Random Access Memory)이 자주 사용됨 
    3) 일반적으로 프로그램을 실행하면, 프로그램은 메인 메모리에 적재(프로세스 형태)됨
    4) 이후에 CPU는 메인 메모리에 적재된 프로그램 명령 코드를 읽어 들여 실행(처리)함.

5. CPU (Central Processing Unit, 중앙처리장치)    
    1) 메인 보드에 가장 먼저 연결하는 장치 중 하나임.
    2) CPU는 컴퓨터의 구성 요소 중에서 인간의 뇌와 같은 역할 수행함.
    3) 연산(computation)과 제어(control)를 담당함 
    4) 산술/논리 장치(Arithmetic Logic Unit, ALU),
        - 산술 연산(+,-,*,/ 등), 논리 연산(AND, OR, XOR 등)
        - 컴퓨터 내에서 매우 빠른 연산 장치임 
       제어장치(Control Unit),
        - 메인 메모리에 적재된 처리해야 할 데이터를 해석하고, 적절한 제어 신호를 생성함 
        - ALU에게 연산 수행을 명령할수 있음 
        - 메인 메모리에 대한 읽기(read) / 쓰기 (write) 등의 제어 명령을 내림 
       레지스터(Register)

6. 프로그램을 실행하면?
    1) 설치된 VS code (보조기억장치,HDD) 실행 => 메인 메모리에 프로그램 적재 => CPU가 실질적인 연산 처리 
    2) 기본적으로 CPU는 계산/제어를 담당함 (ALU 연산 장치 사용)
    3) CPU는 메인 메모리에서 명령어를 읽어, 계산한 뒤에 메인 메모리에 저장함 
    4) 보조기억장치에는 일반적으로 설치된 프로그램이 존재함.

7. 컴퓨터의 일반적인 구조 
    1) 하드웨어(hardware)   
        - 컴퓨터가 동작하도록 해주는 내부 물리적인 장치들 
        - 중앙처리장치 
            - 연산 장치, 제어 장치
        - 기억장치
            - 주기억장치, 보조기억장치
        - 입출력 장치        
    2) 소프트웨어(software)
        - 컴퓨터가 어떻게 동작해야 하는지 명시되어 있는 명령어 집합                 
        - 시스템 SW 
            - 하드웨어 제어 및 관리 
            - 하드웨어 자원에 대한 접근이 용이하며, 대개 저수준 언어로 작성됨 
            - 펌웨어, 운영체제(OS), 컴파일러, 장치 드라이버 등 
        - 응용 SW 
            - 일반적인 사용자 작업 수행을 도움 
            - 컴퓨터 자원에 대한 직접적인 접근이 제한되며, 대개 고수준 언어로 작성됨 
            - 시스템 소프트웨어에 비해 접근하기 쉽고, 친화적이라는 장점이 있음.
            - 웹 브라우저, 게임 등
