Quiz 1. 객체가 가지고 있는 동작을 무엇이라고 하는가?
    - 메서드(Method)

Quiz 2. 하나의 클래스 내에서 동일한 이름의 메서드를 여러 개 사용할 수 있는 
        객체지향 기법을 무엇이라고 하는가?
    - 메서드의 오버로딩 

Quiz 3. 메서드의 시그니처 2가지를 쓰시오 
    - 매개변수의 타입, 개수 

Quiz 4. 정적 바인딩(static binding)이란 무엇인가?
    - 컴파일 시점에서 사용될 메서드가 결정되는 바인딩 

Quiz 5. 메서드의 매개변수 전달 기법 중 자주 사용되는 2가지를 쓰시오 
    - Call by Value, Call by Reference

Quiz 6. 메서드의 수행결과를 반환하는 명령어를 쓰시오. 
    - return
