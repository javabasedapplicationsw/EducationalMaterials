1. 반복문 (loop) for, foreach
    1) 일반적으로 반복하는 횟수가 정해진 경우에 사용 
    2) 초기식, 조건식, 증감식으로 구성되고, 세미콜론(;)으로 구분
        - 초기식은 처음 한 번만 수행
        - 이후, 조건식->반복할 문장->증감식 이 반복적으로 수행됨 
    3)
               ①      ②      ④      
        for(초기식; 조건식; 증감식) {
            반복할 문장;③
        }

        - ①, ②③④  ②③④ ②③④ ...
        - 조건식이 거짓이면 for문 탈출


        foreach문 (향상된 for문)
        
        for( 변수 : 컬렉션,배열 ) {
            반복할 문장;
        }

2. 중첩 반복문 (여러 번 포개짐)
    1) 반복문 내부에 또 다른 반복문이 있음
    2) 여러 겹으로 반복문을 겹쳐서 구현 가능 
    3) 외부 반복문과 내부 반복문 간의 변수 값 변화에 유의하여 구현해야 함     

3. for문 문장들 생략가능함
    1) 초기화식 생략
        - 이미 이전에 값이 초기화 되어 
          for 내부에서 값을 지정할 필요가 없는 경우  
        - 
            int i = 0;
            for( ; i < 5; i++) {

            }           
    2) 조건식 생략 
        - 반복 수행에 대한 조건이 수행문 내부에 있는 경우 
        - 
            for(i = 0; ; i++) {
                sum += i;
                if(sum > 200) break;
            }
    3) 증감식 생략
        - 증감식에 대한 연산이 복잡하거나 다른 변수의 연산 결과값에 결정되는 경우
        - 
            for(i = 0; i < 5; ) {
                ...
                i = (++i) % 10;
            }                 
    4) 무한 반복 
        -
            for( ; ; ) {
                ....
            }
                        
4. break문 사용하기 
    1) 중간에 멈추는 기능
    2) 감싸고 있는 제어문의 블록을 빠져 나오는 기능 (switch문에서도 사용)
    3) 반복문에서는 조건문(if문)과 같이 사용하여 
       조건에 해당되는 경우 반복 수행을 멈추고 반복문 외부로 수행이 이동 
    4) 여러 반복문이 중첩되어 있는 경우엔 break문이 포함되어 있는 반복문만 빠져 나옴 

5. continue문 사용하기 
    1) 무시하고 계속 진행함 
    2) 반복문 내부에서 조건문(if)과 같이 사용하며,
       조건이 맞는 경우 이후 반복문 블럭 내부의 다른 수행문들을 수행하지 않음

6. 반복문 - while문 
    1) 주어진 조건이 참(true)인 동안 지정된 수행문을 반복수행하는 제어문 
    2) 조건이 맞지 않으면 반복하던 수행을 멈추게 됨 
        - 조건은 주로 반복 횟수나 값의 비교의 결과에 따라 true, false 판단 됨 
    3) 예
        - 달리는 자동차, 일정 횟수 만큼 돌아가는 나사못, 특정 온도까지 가동되는 에어컨        

    4) 
        초기식;①             
        while(조건식②) {                    // 조건식이 거짓이면 탈출
            반복할 문장;  ③
            증감식;
        }        

7. 반복문 - do-while문
    1) 조건과 상관없이 한번은 수행문을 수행 
    2) while문은 조건을 먼저 체크하고 반복 수행이 된다면,
       do-while은 조건과 상관없이 수행을 하고나서 조건을 체크 
    3)
        초기식;①
        do {
            반복할 문장;        ②
            증감식;
        } while(조건식);        ③       // 조건식이 거짓이면 탈출
