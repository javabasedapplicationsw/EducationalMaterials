1. 데이터 종류 
    1) 수치형 (사칙연산 가능한 데이터)
        - 연속형 (키, 몸무게)
            - 실수
        - 이산형 (과일개수, 책페이지수)
            - 정수

    2) 범주형
        - 순서형 (랭킹, 학점)
        - 명목형 (성별, 우편 번호)

2. 탐색적 데이터 분석 (그래프)        
    1) Maplotlib (맷플롯립)
    2) Seaborn (시본)

3. 수치형 데이터 시각화 
    1) 데이터가 어떻게 분포되어 있는지가 중요     
        - 고르게 퍼져 있는지, 특정 영역에 몰려 있는지 파악 
        - 어떻게 변환할지, 어떻게 해석해서 활용할지 판단할수 있음 

    2) 히스토그램 (histplot)
        - 구간별 빈도수 나타내는 그래프 

    3) 커널밀도추정 함수 그래프 (kdeplot)
        - 히스토그램을 매끄럽게 곡선으로 연결한 그래프 

    4) 분포도 (displot)
        - 분포를 나타내는 그래프     

    5) 러그플롯 (rugplot)
        - 주변 분포를 나타내는 그래프 
        - 주로 다른 분포도 그래프와 함께 사용 
            - 단일 피처가 어떻게 분포돼 있는지, x축 위에 작은 선분(러그)으로 표시함 
            - 값이 밀집돼 있을수록 작은 선분들도 밀집돼 있음                     

4. 범주형 데이터 시각화 
    1) 막대 그래프 (barplot)
        - 범주형 데이터 값에 수치형 데이터 값이 어떻게 달라지는지 파악할 때 사용함 
            - 평균, 신뢰구간을 그려줌 
            - 수치형 데이터 평균은 막대 높이로 표현, 신뢰구간은 오차 막대(error bar)로 표현함 
        - x 파라미터에 범주형 데이터, y 파라미터에 수치형 데이터를 전달 

    2) 포인트플롯(pointplot)
        - 그래프를 점과 선으로 나타냄 

    3) 박스플롯(boxplot)
        - 5가지 요약 수치 제공                     
            - 최소값
            - 제1사분위 수(Q1)  : 전체 데이터 중 하위 25%에 해당하는 값
            - 제2사분위 수(Q2)  : 50%에 해당하는 값(중앙값)
            - 제3사분위 수(Q3)  : 상위 25%에 해당하는 값
            - 최대값 
        - 이상치
        - 범주별 5가지 요약 수치를 한눈에 파악

    4) 바이올린플롯(violinplot)
        - 박스플롯 + 커널밀도추정 함수 그래프를 합쳐 놓은 그래프 
        - 수치형 데이터의 전체적인 분포 양상 파악        

    5) 카운트플롯(countplot)
        - 범주형 데이터의 개수 확인할 때 사용하는 그래프 

    6) 파이 그래프 (pie)
        - 범주형 데이터별 비율을 알아볼때 사용                 
        
5. 데이터 관계 시각화 
    1) 여러 데이터 사이의 관계를 살펴보기 위한 그래프 
        - 히트맵(heatmap), 라인플롯, 산점도, 회귀선 

    2) 히트맵 
        - 비교해야 할 데이터가 많을 때 주로 사용 
        - flights
            - 범주형 데이터 2개(year, month), 수치형 데이터 1개(passengers)     

    3) 라인플롯 (lineplot)
        - 두 데이터 사이의 관계를 나타낼 때 사용 
        - 전달한 값의 평균과 95% 신뢰구간 나타냄 

    4) 산점도 
        - 두 데이터의 관계를 점으로 표현하는 그래프   

    5) 회귀선을 포함한 산점도 (regplot) 
        - 산점도와 선형 회귀선을 동시에 그려주는 그래프 
        - 회귀선을 그리면 전반적인 상관관계 파악이 좀 더 쉬움 
        - 선형 회귀선과 주변 음영은 95% 신뢰구간 표시
