1. 소개 
    1) KNN 모델 
        - KNearestNeighbors
        - 종속변수가 범주형임
        - 종속변수 개수가 3개 이상인 다중분류(Multiclassification)
    2) K-최근접 이웃 
        - 거리 기반 모델 
        - 각 데이터 간의 거리를 활용해서 새로운 데이터를 예측하는 모델 
        - 가까이에 있는 데이터를 고려하여 새로운 데이터를 예측하는 모델 
            - 이때 가까이에 있는 데이터를 고려하여 예측값이 결정됨    
            - K개의 가장 가까운 이웃 데이터에 의해 예측된다는 의미임  
        - 데이터 간의 거리를 활용해서 새로운 데이터가 어디에 속하는지 구분하는 알고리즘

7. 스케일링 (scale)
    1) 표준화 스케일링 (Standard Scaling)
        - 평균은 0이 되고, 표준편차가 1이 되도록 데이터를 고르게 분포시키는데 사용
    2) 로버스트 스케일링 (Robust Scaling)   
        - 데이터에 아웃라이어가 존재하고, 그 영향력을 그대로 유지하고 싶을 때 사용 
    3) 최소-최대 스케일링 (Min-Max Scaling)
        - 데이터 분포의 특성을 최대한 그대로 유지하고 싶을 때 사용 
    4) 정규화 스케일링 (Normalizer Scaling)
        - 행 기준의 스케일링이 필요할 때 사용하나, 거의 실제로 사용하지 않음

10. 용어 정리
    1) K-최근접 이웃
        - 거리 기반으로 데이터를 분류하는 알고리즘 
        - 3개 이상의 분류도 가능함 
    2) 아웃라이어 
        - 평균치에서 크게 벗어나는 데이터를 의미함 
    3) 스케일링
        - 독립변수의 범위를 동일한 수준으로 만드는 데 사용되는 방법 
            - 최소-최대 스케일링
                - 데이터 분포의 특성을 최대한 그대로 유지하고 싶을 때 사용 
    4) 결측치 
        - 데이터가 비어있는 값
        - NaN, NA, Null 등으로도 표현함
