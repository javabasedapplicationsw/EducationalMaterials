1. 인공지능 > 머신러닝 > 딥러닝 관계

    1) 인공지능이 가장 큰 개념임 
        - 인간의 뇌를 모방하는 모든 프로그램 
        - 자율 주행, 알파고, 쇼핑몰 추천 알고리즘 등이 모두 인공지능에 해당
    2) 머신러닝 (Machine Learning, ML)
        - 기계학습 

        - 비교
            - 전통적인 프로그램 
                - 예측 
                    데이터                -------->  
                                                        컴퓨터   -------->  결과 
                    인간이 작성한 모델     -------->

            - 머신 러닝 
                - 모델 생성
                    샘플 데이터           --------->
                                                        컴퓨터   -------->  모델        
                    정답값                ---------> 

                - 예측 
                    새로운 데이터         ---------->   
                                                        컴퓨터  ----------> 결과
                    모델                 -----------> 

        - 입력값과 해당 정답값에 대한 입력을 해주면,  입력값과 정답값 간의 관계를 
          찾아서 새로운 프로그램(모델)을 만듦  
        - 그 후 새로운 프로그램(머신러닝 모델)에 데이터를 입력해서 값을 예측함  
        - 새로운 데이터가 주어졌을 때 결과를 예측하는 일련의 과정임                   

    3) 딥러닝 (Deep Learning)
        - 심층 학습 
        - 인공 신경망을 기반으로 한 특수한 머신 러닝 기법 
            - 인간의 두뇌 작동 방식을 본떠 개발된 것
        - 빅데이터 기술과 하드웨어 발전, 새로운 알고리즘 등장으로 각광받게 되었음 
        - 자연어 처리나 이미지, 비디오 분석 같은 목적으로 사용됨    

        - 입력층과 출력층 사이에 은닉층(보통 하나 이상의 수많은 은닉층이 존재)을 두어 
          인간의 신경망처럼 작동시킴 (인공신경망) 

        - 딥러닝은 머신러닝 방법 중 하나임 
            - 머신러닝이 딥러닝을 포함 할 수 있음    

2. 머신러닝 기법 
    1) Supervised Learning (지도 학습)
       Unsupervised Learning (비지도 학습)
       Reinforcement Learning (강화 학습)


    2) 
            분류 -----------                                      --------- 군집화
                           |                                     |         
                        지도 학습 --------- 머신 러닝 -------- 비지도 학습       
                           |                                     |
            회귀------------                                      --------- 변환        

    3) 지도 학습 
        - 입력값과 그에 대한 정답을 사용해 이 둘의 관계를 분석하고 예측 모델을 만들어냄 
        - 자동차의 특성(연식, 주행거리, 배기량, 연료 타입 등)과 가격을 입력해 
          중고차 가격을 예측하는 모델을 만듦,
        - 해당 모델로 자동차의 특성이 입력되었을 때 가격을 예측하는 문제 풀이 과정임. 
        - 타겟 y가 명확하게 존재하는 경우 사용하는 학습 방법
        - 현업에서 만날 수 있는 대다수의 문제가 Supervised

    4) 비지도 학습 
        - 정답이 없는 데이터만으로 학습함 
        - 애초에 정답이 제공되지 않기 때문에 찾으려고 하는 답도 명확하지 않음 
        - 타겟 y는 없고, 독립변수 X만 있는 경우 
        - 예) 신용카드 회사에서 고객 데이터를 가지고 최대한 특징이 비슷한 부류끼리 묶는 행위         

3. 머신러닝 프로세스 
    1) 주어진 문제를 푸는 일련의 과정 
    2) 문제를 풀려면 데이터가 주어져야 함 
    3) 단계  -- 반복 (MLOps, Machine Learning Operations)
        - ① 데이터 수집 
            - 캐글, 오픈엠엘
        - ② 데이터 전처리
            - 데이터에 잘못된 값이 들어 있거나 활용하기 불편하게 정리되어 있음
            - 이런 데이터를 수정 및 정리하고 추가로 만들어내는 과정 
            - 피처 엔지니어링 
            - 현업에서 좋은 성과를 내려면 이 과정은 필수임 
        - ③ 모델 학습 
            - 하이퍼파라미터 (사람이 직접 값을 정해줘야 하는 매개변수)를 조절해 모델의 성능을 
              극대화시키는 과정도 포함됨  
        - ④ 모델 평가 
            - 학습이 완료된 모델을 사용해 예측해보고, 결과를 평가하는 단계 
            - 보통 학습된 모델 여러 개를 준비해 비교해보고 가장 좋은 모델을 선정함 
        - ⑤ 모델 배포 
            - 머신러닝 결과물을 활용하는 단계임 
            - 서비스에 직/간접적으로 활용, 예측 결과물을 대시보드 형태로 시각화해 공개 

    4) 데이터는 수시로 바뀔수 있어서 한 번 배포한 모델이더라도 지속적으로 학습해 사용하게 됨         

4. 머신러닝 핵심 라이브러리 
    1) 넘파이(Numpy)
        - 다차원 배열에 빠른 처리를 지원하는 라이브러리 
    2) 판다스 (Pandas)
        - 넘파일 기반으로 구현한 라이브러리.
        - 넘파이보다 쉽게 읽고 변형할 수 있음
    3) 사이킷런 (Sklearn)     
        - 데이터 분석과 머신러닝 관련 다양한 기능 지원 
        - 데이터 전처리, 모델링, 모델 평가 등 광범위한 분야 지원 
    4) 맷플롯립(Maplotlib)
        - 시각화 라이브러리       
    5) 시본(Seaborn)                       
        - 맷플롯립 기반으로 구현한 라이브러리 
        - 맷플롯립보다 사용이 쉽고 더 깔끔한 결과물을 보여줌 

5. 변수 
    1) 통계에서의 변수          
        - 관찰하려는 대상의 '특성'을 의미함 
        - 예
            - 학생들의 신체적 정보를 활용해 학교 성적을 예측하는 프로젝트라고 가정
                - 키, 몸무게, 체질량지수 등이 변수임 (독립변수)
                - 성적도 하나의 변수임 (종속변수) 

    2) 변수의 구분 
        - 독립변수
            - 예측에 사용되는 재료와 같은 변수들
            - 피처 변수(Feature variable)
            - 피처 
        - 종속변수 
            - 예측을 하려는 대상 변수 
            - 목표 변수(Target variable)                        
            - 타깃 
        - 매개변수 
            - 결과에 영향을 주는 매개변수
