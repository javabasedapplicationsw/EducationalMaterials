1. 코드 묶는 방법
    1) 함수(function)
    2) 객체(object)
        - 변수와 함수를 묶는 방법
    3) 모듈(module)
        - 함수나 객체들을 소스 파일 안에 모은 것        

2. 함수 작성 및 호출하기 
    1) 형식
        def 함수이름(매개변수1, 매개변수2, ...):
            명령문1
            명령문2 

3. 여러 개의 값을 반환할 수 있음

4. 함수의 몸체는 나중에 작성할 수 있음
    1) 헤더만 결정하고 몸체는 나중에 작성함 
        - pass 

5. 파이썬은 인터프리터 언어이기 때문에 함수의 순서가 중요함 
    1) 함수 내에서는 아직 정의되지 않은 함수를 호출 할수는 있음        

6. 디폴트 인수 (default argument) 
    1) 함수의 매개변수가 기본값을 가질 수 있음

7. 키워드 인수 (keyword argument)
    1) 인수의 이름을 명시적으로 지정해서 값을 매개 변수로 전달하는 방법임 

8. 가변 인수 (*)
   가변 길이 키워드 인수 (**)
