1. 파이썬의 특징
    1) 쉽고 간결 
    2) 라이브러리 많고 활용이 쉬움 
        - pip 
    3) 대중적인 언어 
        - AI를 위한 대부분의 라이브러리들은 파이썬 기반으로 하고 있음.
    4) 인터프리터 언어 
        - 자신이 작성한 명령문의 결과를 즉시 볼수 있음     

2. 파이썬으로 하기에 안 좋은 영역
    1) 시스템과 밀접한 프로그래밍 영역 
    2) 모바일 프로그래밍

3. 컴파일러 (인터프리터)
    1) 인간이 프로그래밍 언어로 프로그램 작성하면 
       컴파일러(또는 인터프리터)라고 하는 통역 소프트웨어가 
       프로그램을 기계어로 바꿈 

4. 아나콘다
    1) 인기 있는 라이브러리가 거의 모두 포함된 배포판 
        - pip 
        - numpy
        - pandas
        - sklearn
        - ....

5. 모드 구분
    1) 대화형 모드 (interactive mode)
        - 콘솔에서 문장을 한 줄씩 입력하여 실행
    2) 스크립트 모드 (script mode)
        - 파일을 만들어서 저장한 후에 파이썬 인터프리터가 이 파일을 읽어서 한 번에 전부 실행  
        - 코드가 복잡해지면 인터프리터 모드는 번거롭게 됨
