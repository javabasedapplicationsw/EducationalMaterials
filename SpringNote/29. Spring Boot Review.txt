1. 스프링 부트 (Spring Boot)
    1) 스프링 프레임워크에 톰캣(Tomcat)이라는 서버를 내장 + 여러 기능들 추가함 
        - 스프링 부트는 WAS가 필요없음 
    2) 스프링 부트를 사용하면 보안 공격(SQL 인젝션 등)을 막아주는 코드를 추가 할 필요 없음
    3) 스프링 부트는 설정이 쉬움 

2. src/main/resources
    1) static 디렉토리
        - css, js, jpg, png 파일 등 저장함 
    2) templates 디렉토리  
        - 템플릿 파일 저장함 
        - HTML 형식의 파일 
            - 스프링 부트에서 생성한 자바 객체를 HTML 형태로 출력 할수 있음 
    3) application.properties(yml)     
        - 환경 설정, 환경 변수, 데이터 베이스 설정 등을 이 파일에 저장함
