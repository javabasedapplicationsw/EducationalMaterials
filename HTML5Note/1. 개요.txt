1. 웹 프런트엔드 개발의 핵심 내용 위주로 진행 

2. 프런트엔드 개발
    1) HTML, CSS, JS를 사용해 데이터를 그래픽 사용자 인터페이스(GUI)로 변환하고,
       그것으로 사용자와 상호 작용할 수 있도록 하는 것 

    2) HTML (Hyper Text Markup Language)     ==> 기획자 
        - 페이지의 제목, 문단, 표, 이미지, 동영상 등 웹의 구조를 담당 

    3) CSS (Cascading Style Sheets)          ==> 디자이너    
        - 실제 화면에 표시되는 방법을 지정해 콘텐츠를 꾸며주는 시각적인 표현(정적)을 담당
        - 색상, 크기, 폰트, 레이아웃 등 

    4) JS (JavaScript, ECMAScript)                       ==> 개발자 
        - 콘텐츠를 바꾸고 움직이는 등 페이지를 동작시키는 동적 처리를 담당         

3. 웹 표준 
    1) Web Standard
    2) 웹에서 사용되는 표준 기술이나 규칙 
    3) W3C의 표준화 제정 단계의 권고안(REC)에 해당하는 기술         

4. 크로스 브라우징 (Cross Browsing)
    1) 조금은 다르게 구동되는 여러 브라우저에서,
       동일한 사용자 경험(같은 화면, 같은 동작 등)을 줄 수 있도록 제작하는 기술, 방법.

    2) 창 (Window)  
       탭 (Tab)
       주소장 (Address bar)     
       뷰포트 (Viewport)

5. 웹 접근성 
    1) 팀 버너스 리(웹 창시자)
        - 웹의 힘은 그것의 보편성에 있다. 장애에 구애 없이 모든 사람이 접근할 수 있는 것이 필수적인
          요소이다.

6. 웹 이미지 
    1) 비트맵(Bitmap)
        - 픽셀이 모여 만들어진 정보의 집합
        - .jpeg, .gif, .png 
            - JPG(JPEG)
                - 압축률이 훌륭해 사진이나 예술 분야에서 많이 사용
                - 손실 압축 
                - 가장 널리 쓰이는 이미지 포맷 
            - PNG
                - 비손실 압축 
                - W3C 권장 포맷 
            - GIF 
                - 비손실 압축 
                - 여러 장의 이미지를 한 개의 파일에 담을 수 있음 (애니메이션)
                - 8비트 색상만 지원 
            - WEBP 
                - 구글이 개발한 이미지 포맷 
                - 완벽한 손실/비손실 압축 지원 
                - GIF 같은 애미메이션 지원                                    

        - 확대/축소 시 계단 현상, 품질 저하 
    2) 벡터(Vector)
        - 수학적 정보의 형태로 이루어진 이미지 
            - 점, 선, 면의 위치, 색상 등 
        - .svg                              
        - 확대/축소에서 자유로움, 용량 변화 없음 
        - css와 js로 제어 가능
        - 파일 및 코드 삽입 가능 

7. 특수 문자 용어 
    1) `
        - 백틱, 그레이브(Backtick, Grave)
    2) ~
        - 틸드(Tilde), 물결 표시                
    3) !
        - 엑스클러메이션(exclamation mark), 느낌표 
    4) @ 
        - 앳(at sign), 골뱅이 
    5) # 
        - 샵(Sharp), 넘버(Number sign), 우물 정 
    6) ^ 
        - 캐럿(Caret)
    7) &
        - 엠퍼센드(Ampersand)
    8) * 
        - 에스터리스크(asterisk), 별표 
    9) -
        - 하이픈(Hyphen), 대시(Dash), 마이너스 
    10) _
            - 언더스코어(Underscore), 로대시(Low dash), 밑줄 
    11) =
            - 이퀄(Equals sign), 동등  
    12) "
            - 퀘테이션(Quotation mark), 큰 따옴표 
    13) '
            - 아포스트로피(Apostrophe), 작은 따옴표 
    14) :
            - 콜론(Colon)
    15) ;
            - 세미콘론(Semicolon)
    16) .
            - 피리어드(Period), 닷(dot), 점, 마침표 
    17) ?
            - 퀘스천(Question mark), 물음표 
    18) /
            - 슬래시(Slash)
    19) |
            - 버티컬 바(Vertical bar)
    20) \
            - 백슬래시(Backslash), 역 슬래시 
    21) ()    
            - 퍼렌서시스(Parenthesis), 소괄호, 괄호 
    22) {}   
            - 브레이스(Brace), 중괄호 
    23) []
            - 브래킷(Bracket), 대괄호 
    24) <>
            - 앵글 브래킷(Angle Bracket), 꺽쇠괄호
